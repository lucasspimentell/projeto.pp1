Descrição do sistema

O sistema proposto é um e-commerce dedicado à venda de jogos de todos os gêneros.

É o site do e-commerce desenvolvimento 100% na liguagem de HTML, com a seguinte estrutura:

  1. Tela de Login
  2. Tela de formulário para Novo Castrato
  3. Tela de Redefinir senha
  4. Uma tela principal
    • Produtos à venda
    • Subtópicos
    • Carrinho de Compras
    • Sobre do sistema
  5. Detalhes dos produtos
  6. Tela de Finalizar Compra
