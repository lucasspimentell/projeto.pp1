Descrição do sistema

O sistema proposto é um e-commerce dedicado à venda de livros de todos os gêneros, com foco na oferta de obras de alta qualidade. Desenvolvido para atender tanto os amantes da leitura quanto os aficionados por livros raros e exclusivos, o sistema oferece uma ampla variedade de títulos cuidadosamente selecionados para atender a diversos interesses e preferências.

É o site do e-commerce desenvolvimento 100% na liguagem de HTML, com a seguinte estrutura:

  1. Tela de Login
  2. Tela de formulário para Novo Castrato
  3. Tela de Redefinir senha
  4. Uma tela principal
    • Produtos à venda
    • Subtópicos
    • Carrinho de Compras
    • Sobre do sistema
  5. Detalhes dos produtos
  6. Tela de Finalizar Compra
